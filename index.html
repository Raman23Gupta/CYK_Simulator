Core Concepts
📝

Context-Free Grammar (CFG)

A set of recursive rewriting rules used to generate patterns of strings. CFGs are widely used in linguistics and computer science to describe the structure of programming languages.

S ➔ NP VP
NP ➔ Det N
VP ➔ V NP
⚙️

Chomsky Normal Form

The CYK algorithm requires the grammar to be in Chomsky Normal Form (CNF). In CNF, all production rules must be strictly of the form: a non-terminal producing either exactly two non-terminals or exactly one terminal.

A ➔ B C (Two non-terminals)
A ➔ a    (One terminal)
📊

Dynamic Programming

CYK is a dynamic programming algorithm. It works bottom-up by breaking the problem into smaller subproblems. It parses the string by filling a triangular table, calculating which variables can produce each substring.

Time Complexity: O(n³·|G|)
Space Complexity: O(n²·|G|)
